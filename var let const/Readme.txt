var, let and const

These are ways to declare variables in JavaScript.

- var: old way, can be redeclared and updated, but it has scope issues.
- let: can be updated, but cannot be redeclared in the same scope.
- const: cannot be updated or redeclared.

Example:
let name = "John";
const pi = 3.14;

Note: Avoid using var in modern JavaScript.
